ЛР8 — готовый проект Visual Studio

Как открыть:
1. Распаковать архив.
2. Открыть файл L8.sln.
3. Запустить Ctrl+F5.

В проекте один main.cpp с меню:
1 — перегрузка >> и << для класса Stroka.
2 — Money и шаблонные функции.
3 — шифратор через ifstream/ofstream.
4 — состояния потоков good/fail/eof/clear.

Для пунктов 3 и 4 можно использовать test.txt.
Для шифратора пример:
Input file: test.txt
Output file: encrypted.txt
Key: secret

Для расшифровки:
Input file: encrypted.txt
Output file: decoded.txt
Key: secret
